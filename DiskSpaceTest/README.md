## DiskSpaceTest

### Usage
```
DiskSpaceTest.exe

DiskSpaceTest.exe [MaxRange(Megabyte) (Defalut:1024)]
DiskSpaceTest.exe 512

DiskSpaceTest.exe [Output File Path (Defalut: current path)] [Max Number Entry (Defalut:30)] [MinRange(Megabyte) (Defalut:1024)] [MaxRange(Megabyte) (Defalut:1024)]
DiskSpaceTest.exe "C:\" 50 128 512

```
